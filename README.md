# TKFMaa

基于 [MaaFramework](https://github.com/MaaXYZ/MaaFramework) 的「天下布魔」日常自动化助手

</div>

## 安装与使用

1. 下载最新 Release 压缩包并解压（只测试过windows端）
2. 运行 `DependencySetup_依赖库安装_win.bat`
3. 运行 `MFAAvalonia.exe`

## 主要功能

- [x] 启动游戏
- [x] 商城购买
- [x] 领取体力
- [x] 炼金订单
- [x] 每日派遣
- [x] 消耗体力（随缘更新~）
- [x] 调教
- [x] 全境征才
- [x] 奖励领取
- [ ] 协会奖励领取 — 暂未实现，请手动领取
